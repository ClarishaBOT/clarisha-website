Introduction

Clarisha is the perfect music bot to spice up your server and let your member listen to music.

About this Bot

Clarisha Play music in VC with the highest quality and comes with; DJ-Role system, many Filters for your music, easy to use, and have some fun commands!

Technology

The bot is bulit with javascript using the discord.js library, same with the API the bot uses.

The bot uses youtube api to provide music on vc.

Getting Started

To get started with our bot is fairly easy and we have done our best to make it simpler for you.

In-bot help

You can use /help command to see all commands of Clarisha

Adding the bot

To add the bot to your server click here.

This does needs you to either be the owner of the server or have "manage server" permissions.

NOTE: The server owner will get a direct dm to the bot if the owner dm is open, and also Clarisha will send a message to the channel available to chat after adding.

Commands

  • [ ] : Optional Argument
  • ( ) : Command Aliases
